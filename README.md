# 六爻数理占卜 · AstrBot 插件

基于数字起卦的六爻占卜插件，输入数字自动起卦，基于《增删卜易》《易冒》评分体系，输出可视化评分卡片。

## 功能

- 🔢 **数字起卦** — 输入一组数字，自动拆分求和定上下卦，结合时辰定动爻
- 📜 **完整六爻排盘** — 六兽、六亲、地支、世应、动爻、变卦一应俱全
- 📊 **量化评分** — 月建:日辰:动爻 = 6:2:1 古籍权重，总分直观
- 🎯 **用神匹配** — 关键词自动匹配用神（父母、官鬼、妻财、兄弟、子孙）
- 🔄 **回头生克** — 变爻回头生动爻/克动爻自动判定，有根才生效
- ⚡ **刑冲害** — 地支刑、冲、害、合数据齐全
- 🖼️ **卡片输出** — 1800px 高清图片，评分、吉凶、建议、成因一目了然

## 使用方法

在聊天中通过机器人指令调用，格式：

```
@机器人 数字 问题
```

例如：

```
@机器人 [6,4,2] 学业
@机器人 [6,6,4] 买基金能赚吗
@机器人 [9,3,6] 比赛能不能拿奖
@机器人 [1,5,6] 族战能赢吗
```

### 起卦规则

- 数字个数为偶则平分为二，为奇则前少后多
- 前一半之和 % 8 = 上卦
- 后一半之和 % 8 = 下卦
- (上卦数 + 下卦数 + 时辰数) % 6 = 动爻

## 评分体系

参考《增删卜易》《易冒》，权重比：**月建:日辰:动爻 = 6:2:1**

| 状态 | 月建 | 日辰 | 动爻 |
|------|:----:|:----:|:----:|
| 同(旺) | +6 | +2 | 合+1 |
| 生(相) | +4 | +1 | 生+1 |
| 无关 | 0 | 0 | - |
| 克(死) | -3 | -1 | 克-1 |

- **回头生**: 变爻生动爻，+2分（需用神有根）
- **回头克**: 变爻克动爻，-2分
- **大吉** ≥ 6分 | **吉** ≥ 3分 | **平** ≥ -1分 | **凶** ≥ -4分 | **大凶** < -4分

## 配置（省 token）

在 AstrBot 后台 → 插件管理 → 六爻数理占卜 → 设置，可调整以下参数：

| 参数 | 说明 | 建议值 |
|------|------|--------|
| `card_enabled` | 是否生成卡片图片。关闭则只返回文字 | 群聊开，私聊可关 |
| `detail_level` | 精简=只出分数，标准=+成因，完整=+排盘 | 标准 |
| `max_history` | 保留最近几次记录到上下文，0为不保留 | 3 |
| `daily_limit` | 每日最大调用次数，0为不限 | 0 |

省 token 建议：`detail_level=精简` + `card_enabled=false`，每次调用约省 60% token。
